Koala-BR8 made in Gamemaker

Project Description

Koala-BR8 is a puzzle game where you control multiple koalas and you must get each of them to the exit without losing a single one. The catch is that all the koalas move at the same time, you can not control just a single koala, so if you want to move one downwards they will all move downwards.

  • Features explosive traps and moving sawblades
  • Has unique sound effects for colliding with traps and has background music
  • Progressing through the game also increases the difficulty of the puzzles
  • Features animated sprites
